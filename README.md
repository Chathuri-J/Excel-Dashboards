 Project Overview
This project is an **interactive Excel dashboard** built using Pivot Tables, Slicers, and Charts to analyze **bike purchase behavior** across different demographics.  
It helps identify **which customer groups are more likely to purchase bikes** and how various factors such as income, education, region, and commute distance affect their buying decisions.
